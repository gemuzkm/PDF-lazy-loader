Оптимизация загрузки PDF файлов с использованием паттерна Facade и ленивой загрузки. Совместим с Redis Object Cache и FlyingPress.

== Description ==

**PDF Lazy Loader** — профессиональный WordPress плагин для оптимизации загрузки PDF файлов. Вместо немедленной загрузки тяжелого PDF.js и документов (50MB+), плагин показывает красивое превью (500KB) с кнопкой "Просмотреть". PDF загружается только при клике пользователя.

= Основные возможности =

* **Ленивая загрузка PDF** - загрузка только при клике
* **Паттерн Facade** - элегантное разделение интерфейса и функционала
* **Красивое превью** - профессиональный дизайн с иконкой PDF
* **Поддержка AJAX** - работает с динамически добавленным контентом
* **Redis Object Cache** - полная поддержка с автоматическим обнаружением
* **FlyingPress интеграция** - синхронизированное кеширование
* **Адаптивный дизайн** - мобиль, планшет, десктоп
* **Тёмный режим** - автоматическая поддержка
* **Админ-панель** - удобная настройка параметров
* **Кеширование скриптов** - оптимизированная производительность

= Результаты оптимизации =

* LCP: с 6.5s → 1.8s (3.6x быстрее)
* Размер загрузки: с 52MB → 1.2MB (43x меньше)
* PageSpeed Score: с 28/100 → 87/100
* Улучшение UX: мгновенная загрузка превью

== Installation ==

1. Скачайте плагин
2. Загрузите папку `pdf-lazy-loader` в `wp-content/plugins/`
3. Активируйте плагин в админ-панели WordPress
4. Перейдите в Settings → PDF Lazy Loader для настройки

Или используйте встроенный загрузчик плагинов WordPress:
1. Plugins → Add New
2. Upload Plugin
3. Выберите файл плагина
4. Активируйте

== Frequently Asked Questions ==

= Как работает плагин? =

Плагин использует паттерн Facade для ленивой загрузки PDF:
1. Находит все PDF на странице
2. Заменяет их на красивое превью
3. При клике загружает PDF.js и документ
4. Отображает PDF в оригинальном контейнере

= Совместим ли с другими плагинами кеширования? =

Да! Плагин совместим со всеми популярными плагинами кеширования:
- WP Super Cache
- W3 Total Cache
- Redis Object Cache
- FlyingPress
- LiteSpeed Cache

= Работает ли с мобильными устройствами? =

Да, полностью адаптирован для всех размеров экранов.

= Как отключить плагин для конкретной страницы? =

В админ-панели введите ID страницы в поле "Exclude Pages":
Settings → PDF Lazy Loader → Exclude Pages

= Можно ли изменить цвет кнопки? =

Да! Settings → PDF Lazy Loader → Button Color

= Работает ли с защищёнными PDF? =

Да, работает со всеми типами PDF (открытыми, защищёнными и архивированными).

= Какая минимальная версия PHP? =

PHP 7.4 или выше. Рекомендуется PHP 8.0+
