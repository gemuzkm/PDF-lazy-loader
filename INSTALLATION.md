# Инструкция по установке PDF Lazy Loader

## Структура файлов

Убедитесь, что структура папок следующая:

```
/wp-content/plugins/pdf-lazy-loader/
├── pdf-lazy-loader.php
├── assets/
│   ├── js/
│   │   ├── admin.js
│   │   └── pdf-lazy-loader.js
│   └── css/
│       └── admin.css
```

## Шаги установки

1. **Загрузите все файлы** в папку `/wp-content/plugins/pdf-lazy-loader/`
   - Если папки `pdf-lazy-loader` не существует, создайте её
   - Убедитесь, что главный файл называется точно `pdf-lazy-loader.php`

2. **Проверьте права доступа**
   - Папки: 755
   - Файлы: 644

3. **Активируйте плагин**
   - Перейдите в WordPress Admin → Плагины
   - Найдите "PDF Lazy Loader"
   - Нажмите "Активировать"

4. **Настройте плагин**
   - Перейдите в Настройки → PDF Lazy Loader
   - Настройте цвета кнопок и другие параметры

## Проверка установки

Если плагин не отображается в списке плагинов:

1. Проверьте, что файл `pdf-lazy-loader.php` находится в папке `/wp-content/plugins/pdf-lazy-loader/`
2. Убедитесь, что в начале файла есть правильный заголовок плагина WordPress:
   ```php
   /**
    * Plugin Name: PDF Lazy Loader
    * ...
    */
   ```
3. Проверьте права доступа к файлам
4. Проверьте логи ошибок WordPress (wp-content/debug.log)
5. Убедитесь, что версия PHP >= 7.2

## Решение проблем

### Плагин не виден в списке
- Проверьте синтаксис PHP: `php -l pdf-lazy-loader.php`
- Убедитесь, что нет ошибок в логах WordPress
- Проверьте, что папка плагина называется `pdf-lazy-loader` (без пробелов)

### Ошибки при активации
- Проверьте версию PHP (требуется >= 7.2)
- Проверьте версию WordPress (требуется >= 5.0)
- Включите отладку WordPress в wp-config.php:
  ```php
  define('WP_DEBUG', true);
  define('WP_DEBUG_LOG', true);
  ```
